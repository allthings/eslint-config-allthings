# eslint-config-allthings

ESlint shareable config for Allthings style

## Setup

```shell
yarn add -DE @allthings/eslint-config
```

## Usage

Add this to your .eslintrc.js

### React projects

```js
module.exports = {
  extends: ['@allthings/eslint-config'],
}
```

## Development

Publishing to npm

```shell
yarn deploy
```
